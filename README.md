# Budget Tracker PWA



 ![Application Image](https://i.gyazo.com/57b3a109b5896dd4d26cb4a1c26edffa.jpg)

## Application Features

* Add/subract funds
* Track expenses using a graph
* Offline expense tracking
* Database syncing from offline transactions

## Built With

* Jquery
* MongoDB
* Javascript 
* Express
* Webpack
* CacheAPI
* IndexedDB
